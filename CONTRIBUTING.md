# Contributing to Nepali Calendar SDK

Thanks for your interest! Bug reports, fixes, docs improvements, and feature ideas are welcome.

## How to Contribute

1. **Report issues**  
   Open an issue with details (bug description, expected/actual behavior, code example if possible).

2. **Submit changes**
    - Fork the repo
    - Create a branch: `git checkout -b fix/something` or `feature/new-thing`
    - Make changes + add/update tests (`src/test/`)
    - Run tests: `mvn test`
    - Commit clearly (e.g., "fix: invalid BS day handling")
    - Push and open a Pull Request to `main`

## What We Welcome
- Bug fixes (especially date edges/leap years)
- Range extensions (beyond 2085 BS)
- Formatting improvements (Nepali numerals, Devanagari)
- Documentation tweaks

Questions? Open an issue — happy to discuss!
